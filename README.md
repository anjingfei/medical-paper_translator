# medical-paper_translator
A beginner-friendly web project for practicing clinical medical literature translation. Designed by a newcomer with zero programming experience—please forgive any imperfections and provide feedback for improvement.
# 医学文献翻译练习网站

本项目是一个面向医学英语文献翻译学习的网页练习工具，适合用于医学专业学生、临床医学学习者以及有医学文献阅读需求的用户进行翻译训练。

## 在线访问

👉 [点击这里进入医学文献翻译练习网站](https://anjingfei.github.io/medical-paper_translator/)

## 项目内容

本网站包含医学文献翻译练习内容，帮助用户通过短篇医学英文材料进行阅读、理解和翻译训练。
